# 🥊 GlovesOn - Boxing Training Journal

**GlovesOn** არის Flask-ზე აწყობილი ვებ-აპლიკაცია, რომელიც შექმნილია კრივით დაკავებული სპორტსმენებისთვის, რათა მათ მარტივად შეძლონ თავიანთი ვარჯიშების ციფრული კონტროლი და თვალყურის დევნება.

---

### მთავარი ფუნქციონალი:

1. **მომხმარებელთა სისტემა**
   * მომხმარებლების რეგისტრაცია და ავტორიზაცია უსაფრთხოების ნორმების დაცვით.
   * პროფილის გვერდი, სადაც ინტეგრირებულია **DiceBear API** ავტომატური ავატარების გენერირებისთვის, ასევე საკუთარი ფოტოს ატვირთვის ფუნქცია.

2. **ვარჯიშების ჟურნალი (Training Journal):**
   * სრული **CRUD ფუნქციონალი** (Create, Read, Update, Delete).
   * ბოქსიორებს შეუძლიათ დალოგონ ყოველი ვარჯიში: ჩაწერონ ტიპი, ინტენსივობა, რაუნდების რაოდენობა და თარიღი.

3. **Like/Dislike სისტემა (Training Day Evaluation):**
   * ინტერაქტიული სისტემა, რომელიც მუშაობს მონაცემთა ბაზასთან ერთად.
   * მომხმარებელს შეუძლია შეაფასოს თავისი ვარჯიშის დღის სრულყოფილება: თუ დათქმული ვარჯიში ბოლომდე შეასრულა — წერს Like-ს, წინააღმდეგ შემთხვევაში — Dislike-ს.

---

### ტექნოლოგიური სტეკი & არქიტექტურა (Clean Code):

* **Backend:** Python / Flask
* **Database:** SQLite (მართვა ხდება `instance` საქაღალდიდან, რომელიც უსაფრთხოების მიზნით შეყვანილია `.gitignore`-ში)
* **Frontend:** HTML5, CSS, Bootstrap (Responsive დიზაინი)
* **Dependencies:** ყველა საჭირო ბიბლიოთეკა გაწერილია `requirements.txt` ფაილში.
